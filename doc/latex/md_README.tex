The Raytracer project is a collaborative effort aiming to create a ray tracing engine using modern programming techniques. ~\newline
 The project focuses on implementing various features such as different materials, lighting models, and geometries. ~\newline
 This project was carried out by \href{mailto:jade.magnin@epitech.eu}{\texttt{ jade.\+magnin@epitech.\+eu}} \href{mailto:nathanael.moehn@epitech.eu}{\texttt{ nathanael.\+moehn@epitech.\+eu}} \href{mailto:fahad.raja-muhammad@epitech.eu}{\texttt{ fahad.\+raja-\/muhammad@epitech.\+eu}} and \href{mailto:nicolas.ferreira-andre@epitech.eu}{\texttt{ nicolas.\+ferreira-\/andre@epitech.\+eu}}.\hypertarget{md_README_autotoc_md1}{}\doxysection{Engine Features}\label{md_README_autotoc_md1}
The ray tracing engine supports various features\+:

Different geometries\+: The engine can render different geometric shapes \+: sphere, plane, cone and cylinder. ~\newline
 Materials\+: The engine supports different materials, each with its own light reflection characteristics like Lambertian, Dielectric or Metal. ~\newline
 Texture \+: The shapes supports different textures like chessboard, image or noise. ~\newline
 Lighting\+: The engine supports different lighting models, including diffuse lighting or directional. ~\newline
